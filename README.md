# ecu-ai-analyzer

Projet de Machine Learning et d'analyse binaire pour fichiers ECU.

## Objectifs

- Détection type ECU
- Détection SW/HW
- Classification Original vs Modifié
- Détection des modifications :
  - DPF OFF
  - EGR OFF
  - SCR OFF
  - Stage 1
- Visualisation des différences binaires

## Technologies

- Python
- Jupyter Notebook
- Scikit-learn
- XGBoost
- NumPy
- Pandas

## Structure du projet

- data/
- notebooks/
- src/
- models/
- reports/
